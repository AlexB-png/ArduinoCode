<h1>Pins and Components</h1>

| Sensor Pin | Arduino Pin |
| ---------- | ----------- |
| TRIG       | D8          |
| ECHO       | D9          |
| VCC        | 5V          |
| GND        | GND         |


| LCD Pin | Arduino Pin |
| ------- | ----------- |
| SDA     | A4          |
| SCL     | A5          |
| VCC     | 5V          |
| GND     | GND         |


| LED Color | Arduino Pin | Notes                      |
| --------- | ----------- | -------------------------- |
| Blue      | D3          | Through 220Ω resistor      |
| Green     | D4          | Through 220Ω resistor      |
| Red       | D5          | Through 220Ω resistor      |
| Common    | GND         | Shared ground for all LEDs |

<h1>Requirements</h1>

<ul>
  <li>1x LCD</li>
  <li>1x Ultrasonic Sensor</li>
  <li>3x 220 ohm resistors</li>
  <li>1x RGB LED</li>
  <li>1x Arduino UNO</li>
  <li>Jumper Cables</li>
</ul>
